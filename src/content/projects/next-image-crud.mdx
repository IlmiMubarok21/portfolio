## Next Image CRUD: Basic Image CRUD with Postgresql and Vercel Blob Storage

Next Image CRUD, a basic image create, read, update, and delete functionality application using postgresql and vercel blob storage.
[Preview](https://upload-brown-rho.vercel.app/).

### Stack

- [React](https://react.dev) - React is a free and open-source front-end JavaScript library for building user interfaces based on components.
- [Next](https://nextjs.org) - Next.js is a React framework for building full-stack web applications
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
