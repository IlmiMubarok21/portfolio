## GRIP Force: Warehouse Management System

GRIP Force - Warehouse Management System (WMS) Application to support the supply chain team business process that powered with React Native and Codepush for Android and IOS. This is a minified version of [GRIP Scale](https://ilmimubarok.my.id/projects/grip-scale).

### Stack

- [Next.js](https://nextjs.org) - Next.js is a React framework for building full-stack web applications.
- [Material UI](https://mui.com) - Material UI is a production-ready component library for building user interface for React.js.
