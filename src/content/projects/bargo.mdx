## Bargo: Human Resource Management System

Bargo - Human Resource Management System (HMR) to manage IT Resources, leads, and customer. Powered with React.js and TailwindCSS on the front-end.

### Stack

- [React.js](https://react.dev) - React is a free and open-source front-end JavaScript library for building user interfaces based on components.
- [TailwindCSS](https://tailwindcss.com) - A utility-first CSS framework packed with classes that can be composed to build any design, directly in your markup.
